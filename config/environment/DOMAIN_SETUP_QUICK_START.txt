📋 STEP 1: ADD DOMAIN IN VERCEL (5 minutes)

1. Open Vercel Dashboard:
   👉 https://vercel.com/mirxa27s-projects/newomen

2. Go to: Settings → Domains

3. Click "Add" button

4. Enter: newomen.me

5. Click "Add" again

6. Copy the DNS records Vercel shows you:
   
   A Record:
   ┌────────────────────────────────────────┐
   │ Type:  A                               │
   │ Name:  @                               │
   │ Value: 76.76.21.21                     │
   └────────────────────────────────────────┘
   
   CNAME Record (for www):
   ┌────────────────────────────────────────┐
   │ Type:  CNAME                           │
   │ Name:  www                             │
   │ Value: cname.vercel-dns.com            │
   └────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 STEP 2: CONFIGURE DNS AT YOUR REGISTRAR (10 minutes)

1. Login to where you bought newomen.me
   (GoDaddy, Namecheap, Cloudflare, etc.)

2. Find "DNS Management" or "DNS Settings"

3. Add A Record:
   ┌────────────────────────────────────────┐
   │ Type:  A                               │
   │ Name:  @ (or leave blank)              │
   │ Value: 76.76.21.21                     │
   │ TTL:   Automatic or 3600               │
   └────────────────────────────────────────┘

4. Add CNAME Record:
   ┌────────────────────────────────────────┐
   │ Type:  CNAME                           │
   │ Name:  www                             │
   │ Value: cname.vercel-dns.com            │
   │ TTL:   Automatic or 3600               │
   └────────────────────────────────────────┘

5. Delete any conflicting A or CNAME records

6. Save changes

7. Wait 1-2 hours for DNS propagation
   (Check status: https://dnschecker.org/#A/newomen.me)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   👉 https://supabase.com/dashboard/project/fkikaozubngmzcrnhkqe/auth/url-configuration

2. Update Site URL:
   ┌────────────────────────────────────────────────────────────────┐
   │ Site URL:                                                      │
   │                                                                │
   │ OLD: https://newomen-dbpuusg87-mirxa27s-projects.vercel.app   │
   │ NEW: https://newomen.me                                        │
   └────────────────────────────────────────────────────────────────┘

3. Update Redirect URLs (remove old, add new):
   
   ❌ REMOVE:
      https://newomen-dbpuusg87-mirxa27s-projects.vercel.app/**
      Any URLs with "dualite"
   
   ✅ ADD:
